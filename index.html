⚙ SETTINGS
Language
Add languages by creating extra columns in your Translations tab in Google Sheets. Each column header becomes an option here.
Default Card Display
Choose how operative cards display by default. Individual cards can still be switched per-item via the camera icon — those overrides reset when you reload the page.
PDF Keyword Glossary
Append a glossary page listing every keyword used across your roster, each shown once with its definition text if your Keywords sheet has one. The Full PDF export also shows the keyword's Drive image if one is linked; the Simple print stays text-only for speed.
Sheet Health Check
Scan your loaded Google Sheet data for missing fields, broken links, duplicate names, and other issues that could cause problems in the app.
Connection Settings
Update your Apps Script URL or add/change your Supabase connection.
Preload for This Session
Fetches every card image now, while you have signal, so browsing and building works with no further network calls. Only lasts for as long as this app stays open — closing or reloading it clears this and you'll need to run it again.
